느낀점 IT 영재캠프 참여 후 느낀점을 이야기해 보았습니다.

  1. Q1디자인 수업 후 느낀점

    박민희: 디자인 수업은 나에게 있어서 유용한 수업인 것 같다.
    배서연 : 디자인 프로그램 '픽슬러'로 잘라내기를 할 때 정말 신기하였고, 다음에도 이런 프로그램을 활용하여 제작물을 만들고 싶다
    임수연 : 생각보다 편집하는게 힘들었다...
    최승빈 : 처음에 엄청 긴장했는데 해 보니 정말 유익하고 관심이 더 생기고 특히 사진 편집 포토샵 비슷한 것이 제일 좋았고 관심이 갔습니다.

  2. Q2SW수업 후 느낀점

    박민희: 코딩 자체는 이번에 처음하는 거라 어려웠지만 재미있었다.
    배서연 : 처음에 늦게와서 살짝 헤멨지만, 프로그래밍 기초적인 것을 하며, 흥미를 느낄 수 있어서 아주 유익한 수업이라는 생각이 들었다.
    임수연 : 뭔가 글씨가 많아서 무서웠지만 그래도 잘 한 것 같아서 다행이다
    최승빈 : 프로그래밍을 하긴했는데 정말 지루했었는데 선생님들께서 정말 지루하지 않게 잘 설명해주셔서 더욱 관심이 갔던것 같습니다.

  3. Q32020 it영재캠프에 참여 후 느낀점

    박민희: 화상 수업으로 캠프를 한 것이 처음에는 정말로 신선했고, 수업도 정말로 재미있었다.
    배서연 : 이를 통해 미림마이스터고에 대한 열망(?)이 더 커졌고, 제작물까지 만듬으로써 뿌듯함을 느낄 수 있는 알찬 캠프였다고 느꼈다.
    임수연 : 캠프가 되게 신선하고 재미있어서 좋았다 이 학교에 입학하고 싶다!!!​
    최승빈 : 학교 과에 대해서 더 자세히 알 수 있는 계기가 되어 정말 유익하고 즐거운 추억이 된 것 같습니다.

  4. Q4캠프에서 배워간 점

    박민희: 이번 캠프에서 편집하기 유용한 픽슬러라는 사이트를 찾아내서 좋았고 코딩은 처음으로 하여서 어렵긴 했지만 재미있었다.
    배서연 : 프로그래밍과 디자인에대한 자신감을 배웠고, 줌 프로그래밍을 활용하여 소통하는 방법도 배워서 아주 좋았다.
    임수연 : 프로그래밍과 디자인을 직접 해보니 배우기 전보다 더 관심이 생겼습니다.
    최승빈 : 프로그래밍을 하는건 봤는데 실제로 해보니 정말 즐겁고 신기하더라고요 정말로 즐거운 캠프였던것 같아요!
